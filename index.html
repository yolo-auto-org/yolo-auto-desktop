Follow these steps. Even if you've never opened a terminal before, you'll be up and running in minutes.

1

Open Your Terminal

Don't worry — this is just a window where you type text commands. Pick your operating system below:

🍎 Mac
🪟 Windows
🐧 Linux

Press ⌘ Cmd + Space to open Spotlight, then type "Terminal" and hit Enter.

Alternatively: Finder → Applications → Utilities → Terminal

Press the Windows key, type "PowerShell" or "Command Prompt", and hit Enter.

💡 Pro tip: Install Windows Terminal from the Microsoft Store for the best experience.

Press Ctrl + Alt + T, or search your apps for "Terminal".

Most Linux distributions also have it in your apps menu under "System Tools".

2

Check You Have Node.js

YOLO Auto needs Node.js (don't worry about what that means — it's just something your computer needs). Let's check if you have it:

$ node --version

If you see a number like v20.11.0, you're golden. Skip to Step 3!

If you see "command not found", install Node.js below:

🍎 Mac
🪟 Windows
🐧 Linux

The easiest way: go to nodejs.org, download the LTS version, and run the installer.

After installing, close and reopen Terminal, then run node --version to confirm.

Go to nodejs.org, download the LTS version, and run the installer. Make sure to check the box that says "Automatically install the necessary tools".

After installing, close and reopen PowerShell/Command Prompt, then run node --version.

$ curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
$ sudo apt-get install -y nodejs

Other distros: check nodejs.org for your package manager.

3

Install YOLO Auto

Copy and paste this one line into your terminal (paste is Ctrl+V or ⌘ Cmd+V on Mac):

$ npm install -g @mariozechner/pi-coding-agent

This downloads and sets up YOLO Auto. It'll take about 30 seconds. Once it finishes, you're almost there!

What just happened? npm (Node's package manager) downloaded the YOLO Auto coding agent and made it available everywhere on your computer. You only need to do this once.

❓ FAQ

Is this really free?
The agent itself is free and open source. You pay the AI provider for API usage — typically a few dollars a month for heavy use. Or use an existing Claude Pro / ChatGPT Plus / Copilot subscription at no extra cost.
Do I need to know how to code?
You don't need to be an expert, but basic familiarity helps. Even if you're new, you can ask it to explain things as it goes. Many beginners use coding agents to learn while building.
What can it actually build?
Websites, APIs, CLI tools, data pipelines, games, mobile apps (React Native), browser extensions — anything you can build with code. It works with any programming language.
Can it break my computer?
It runs commands that can modify files and install software. It's smart about what it runs, but it's a good idea to use version control (Git) and review what it does. If you're nervous, run it in a folder with no important files first.
Windows / Mac / Linux — which is best?
All three work great. On Windows, use Windows Terminal for the best experience. On Mac, the built-in Terminal or iTerm2 are both excellent. On Linux, pretty much any terminal emulator will do.
How does it compare to Cursor / Copilot / Claude Code?
YOLO Auto is more lightweight and terminal-native. It doesn't have a GUI — it's pure terminal. This makes it faster, more flexible, and easier to automate. It's also extremely extensible via TypeScript plugins, prompt templates, and skills.

🔧 Troubles? We Got You

"command not found: pi"
Your terminal doesn't know where npm put it. Try closing and reopening your terminal. If it still doesn't work, add npm's global bin to your PATH (search "npm global bin path" + your OS).
"permission denied" during install
On Mac/Linux, try: sudo npm install -g @mariozechner/pi-coding-agent. On Windows, run PowerShell as Administrator.
API key not working
Make sure you ran the export command in the same terminal window where you're running pi. The key only lasts for that session unless you add it to a startup file.
