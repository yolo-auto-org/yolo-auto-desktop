❓ Questions You Might Have

What exactly can it do for me?
Almost anything on your computer: organize files, rename batches of photos, create spreadsheets, write documents, convert file formats, download and sort things, run programs, search through your files — and a lot more. If you can describe it, it can probably do it.
Do I need technical skills?
Nope! You just talk to it in plain English. It's designed to work for everyone, especially the stuff you'd normally spend hours Googling how to do.
Is it safe? Can it mess up my computer?
You see everything it does as it happens. It's like watching someone use your computer — you can stop it anytime. It's smart about not doing dangerous things, but if you're nervous, start with a throwaway folder. Think of it like letting a friend use your laptop: you're watching, and you can always say "stop."
How is this different from asking ChatGPT?
ChatGPT gives you instructions. YOLO Auto does the work. With ChatGPT, you read the answer, then go do the steps yourself. With YOLO Auto, the assistant is right there on your computer — it opens folders, moves files, creates documents, runs programs. You just watch and approve.

Windows, Mac, or Linux — which one works best?
All three work great. On Windows, grab Windows Terminal from the Microsoft Store (free) for the smoothest experience. Mac's built-in Terminal is perfect. Linux users are already pros at this.

🔧 Stuck? Try These

"command not found: pi"
Your terminal might not know where YOLO Auto is. Try closing the terminal and opening a fresh one. Still not working? Search online for "add npm global bin to PATH" + your operating system.
"permission denied" when installing
On Mac/Linux, try adding sudo in front: sudo npm install -g @mariozechner/pi-coding-agent. On Windows, run PowerShell as Administrator.
Membership key not working
Make sure you typed the export line in the same terminal window where you're launching pi. The key only sticks around for that window unless you add it to a startup file.
